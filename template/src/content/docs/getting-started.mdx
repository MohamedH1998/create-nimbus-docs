<PackageManagers type="run" args="dev" />

Your site is now running at `http://localhost:4321`.

## Project structure

| Path | Purpose |
| --- | --- |
| `src/content/docs/` | Your documentation pages (MDX) |
| `src/docs.config.ts` | Site title, logo, links |
| `src/styles/globals.css` | Theme tokens — colors, fonts, layout |

Edit any `.mdx` file in `src/content/docs/` and the browser reloads automatically.
